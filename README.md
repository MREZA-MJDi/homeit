# 🏠 HomeIT

> Professional On-Site IT Services Platform

HomeIT is a modern web platform that connects customers with professional IT technicians for on-site technical services.

Customers can request services such as:

- Windows installation
- Computer troubleshooting
- Network configuration
- Wi-Fi & modem setup
- Software installation
- Website development
- Technical consulting
- Remote support
